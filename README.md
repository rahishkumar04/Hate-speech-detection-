A machine learning project that detects hate speech in tweets using NLP and Logistic Regression. It preprocesses text, extracts features using TF-IDF, and classifies tweets as hate or non-hate.
Features
Text preprocessing using NLTK
TF-IDF feature extraction
Logistic Regression model
Model evaluation (accuracy, confusion matrix)

Tech Stack: Python, Pandas, NLTK, Scikit-learn, Matplotlib

Usage: python hate_speech_detection.py

Output: 0 → Non-hate speech , 1 → Hate speech


Goal: To automatically detect harmful content in social media text.
